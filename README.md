# homebrew-mac-mp4-screen-rec

Homebrew tap for [`mac-mp4-screen-rec`](https://github.com/arch1904/MacMp4ScreenRec).

## Install

```bash
brew tap arch1904/mac-mp4-screen-rec
brew install mac-mp4-screen-rec
```

## Upgrade

```bash
brew update
brew upgrade mac-mp4-screen-rec
```

## HEAD Builds

To install the latest unreleased version from `main`:

```bash
brew reinstall --HEAD mac-mp4-screen-rec
```

## Formula

The formula lives at:

```text
Formula/mac-mp4-screen-rec.rb
```

## Releasing

Stable releases in this tap track tagged releases from the source repo:

- Source repo: `https://github.com/arch1904/MacMp4ScreenRec`
- Tap repo: `https://github.com/arch1904/homebrew-mac-mp4-screen-rec`
